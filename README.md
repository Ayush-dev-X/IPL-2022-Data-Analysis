# 🏏 IPL 2022 Data Analysis & Visualization

A comprehensive Exploratory Data Analysis (EDA) project on the **IPL 2022** season using **Python, NumPy, Pandas, Matplotlib, and Seaborn**. The project explores team performances, player statistics, match outcomes, toss decisions, and venue analysis to uncover meaningful insights from the tournament.

---

## 📌 Project Overview

The objective of this project is to analyze the IPL 2022 dataset and answer cricket-related questions using data analysis and visualization techniques. Through data cleaning, exploration, and visualizations, this notebook highlights key trends and performances throughout the tournament.

---

## 🎯 Project Objectives

- Perform data cleaning and preprocessing.
- Explore IPL 2022 match statistics.
- Analyze team and player performances.
- Study toss and match-winning patterns.
- Visualize important trends using charts.
- Extract meaningful insights from the dataset.

## 📂 Dataset

The dataset contains information about IPL 2022 matches, including:

- Match details
- Teams
- Venue
- Toss winner & decision
- Match winner
- First & second innings scores
- Player of the Match
- Top scorer
- Best bowling figures
- Highest wickets
- Match margins

---

## 📊 Analysis Performed

This project answers questions such as:

- Which team won the most matches?
- Which venue hosted the most matches?
- Which player won the most Player of the Match awards?
- Which batter scored the highest individual score?
- Which bowler recorded the best bowling figures?
- Which team scored the highest total?
- Does winning the toss increase the chances of winning the match?
- How were matches won (runs vs wickets)?

---

## 📈 Visualizations

The notebook includes visualizations to understand:

- Team performance

- Toss analysis
- Match-winning patterns
- Player performance
- Bowling statistics
- Batting statistics

---

## 💡 Key Insights

- Gujarat Titans emerged as the IPL 2022 Champions.
- Winning the toss did not always guarantee victory.
- Several players delivered outstanding batting and bowling performances.
- Different venues showed varying scoring patterns.
- Data visualization helps uncover trends that are difficult to identify from raw data alone.
